# Фильмы

## Описание

Скрипт выводит в терминал список фильмов, которые сегодня в прокате и сортирует их по рейтингу _imdb_

## Требования

*Python3.5*

## Как запустить

Склонировать репозиторий и установить зависимости из requirements.txt:

```sh
git clone https://github.com/Safintim/13_cinemas.git
cd 13_cinemas
pip install -r requirements.txt
```
Запустить скрипт

```sh
python cinemas.py msk
```
